# setup
1. install git https://git-scm.com/install/
2. in a terminal with git, run `git clone https://github.com/pascal-sauer/git-for-students.git`
3. get current path with `pwd`
4. install OpenCode Desktop https://opencode.ai/download
5. open OpenCode Desktop, open project -> select path from step 3
6. OpenCode chat: "read AGENT.md - who are you?" -> should say "I'm a git tutor"
